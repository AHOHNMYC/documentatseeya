bool ScrollMessage::drawUpdate ( const char *  message,
TickType_t  currentTick 
)

Draw and update the scroll message if needed.

This function does not call OLED::refresh(). If this function returns true, the caller shall call OLED::refresh() to draw the modified framebuffer to the OLED screen.

Parameters
messageThe null-terminated message string. This must be the same string as the previous call, unless this ScrollMessage instance is in its initial state or reset() has been called.
currentTickThe current tick as returned by xTaskGetTickCount().
Returns
Whether the OLED framebuffer has been modified.
